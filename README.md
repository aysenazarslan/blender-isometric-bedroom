# 3D Isometric Bedroom - Blender Project

This project is a 3D isometric bedroom scene created in Blender.

## Project Content

The scene includes:
- Bed
- Nightstands
- Wardrobe
- Window
- Carpet
- Ceiling light
- Room floor and walls

## Used Blender Features

- Mesh modeling
- Edit Mode operations
- Extrude
- Inset
- Loop Cut
- Bevel Modifier
- Solidify Modifier
- Subdivision Surface Modifier
- Array Modifier
- Material assignment with Shading nodes
- Area Light and Sun Light
- Camera setup
- Render output
